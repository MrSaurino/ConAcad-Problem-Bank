# ConAcad Problem-Bank (Data Structures & Web development)

This repository houses a comprehensive collection of solved problems from the ConAcad platform, spanning from 2021 to 2026. The solutions cover key academic subjects, including:
* **Web Programming**
* **Data Structures**
* **Programming Fundamentals**

### 🤝 Contributing
Is there a new problem on the platform that isn't here yet? Don't hesitate to contribute! 

We welcome developers and students to open issues or submit Pull Requests. Let's work together to expand this knowledge base and be a guiding light for future students who might be stuck or looking for a little extra help.
